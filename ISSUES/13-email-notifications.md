# Email notifications for signup and reminders

GitHub issue: https://github.com/cratharavi/skills-integrate-mcp-with-copilot/issues/13

**Description**

Integrate an email provider (SMTP or third-party) to send confirmation emails on signup and reminders before activity start.

**Tasks**

- Add email sending utility and configuration
- Trigger confirmation email after signup
- (Optional) Add scheduled reminders before activity start
- Add tests or mocks for email sending

**Acceptance criteria**

- Signup triggers an email with activity details
- Reminder mechanism is config-driven and testable
- Email sending logic is covered by tests/mocks

**Labels**: backend, enhancement
