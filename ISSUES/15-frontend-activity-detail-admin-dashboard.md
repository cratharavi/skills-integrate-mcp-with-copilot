# Frontend: Activity detail page and admin dashboard

GitHub issue: https://github.com/cratharavi/skills-integrate-mcp-with-copilot/issues/15

**Description**

Create frontend pages to view activity details, join activities, and an admin dashboard for managing activities and participants.

**Tasks**

- Add Activity detail page with join/unregister actions
- Add Admin dashboard for activity CRUD and participant lists
- Wire UI to backend endpoints and handle auth
- Add basic styling and responsiveness

**Acceptance criteria**

- Users can view activity details and perform join/unregister from the UI
- Admins can manage activities from the dashboard
- Basic accessibility and responsive layout present

**Labels**: frontend, feature, ux
