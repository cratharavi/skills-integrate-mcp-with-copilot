# Add tests and CI workflow

GitHub issue: https://github.com/cratharavi/skills-integrate-mcp-with-copilot/issues/17

**Description**

Add unit and integration tests and configure GitHub Actions to run tests on PRs and pushes.

**Tasks**

- Add pytest tests for backend endpoints and models
- Add test fixtures and sample data
- Add GitHub Actions workflow to run tests and linting on PRs

**Acceptance criteria**

- Tests run and pass in CI
- Pull requests trigger the test workflow and show results
- Basic coverage for critical endpoints exists

**Labels**: tests, ci
